  5. If you have pirated Minecraft in the past

    1. First, you will need to delete the tool. If you don’t know where it is, try looking in your Downloads folder or any Minecraft folder you have on your computer.
    2. Press Windows Key and R at the same time. In the "Run" box, copy and paste the entire command:

      3. powershell -command "Start-Process notepad $env:windir\system32\drivers\etc\hosts" -Verb runas

    3. A blue window will briefly appear, then a UAC window. Click Yes in the UAC window that pops up. A Notepad window should open with text.
    4. Look for lines that contain the word "mojang" and delete those lines completely.
    5. Save the notepad file. Make sure Notepad does not ask you where to save the file; if that happens, start over and make sure you type the whole command
